## 二、使用说明

   1、在右上角的"添加要做的事"处添加待办事项，按enter键完成添加
   2、点击每个事项右侧的圆形X按钮删除这个事项
   3、点击网页右下角的垃圾桶按钮删除所有的事项
   4、双击事项内容文字进行修改
